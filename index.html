O que é Html?

  • Html siginfica Hyper Text Markup Language
  • Html é a linguagem de marcação padrão para a criação de páginas de web
  • Html descreve a estrutura de uma página de web
  • HTML cosiste em uma série de elementos
  • Os elementos Html informam ao nevegador como exibir o conteúdo
  • Os elementos HTML rotulam partes de conteúdo como "isto é um titulo", "isto é um paragrafo", "isto é um link. e etc."

Introdução ao CSS

Como já vimos ateriormente, as CSS são as Cascading Style Sheets (Folhas de estilos em cascatas). Elas são usadas para configurar um resultado visual ods elementos Html. As configurações das CSS são realizadas através dos seletores. Vamos ver a anatomia de um seletor.

Tipos de CSS

  • Inline- by using style attribute inside Html elements
  • Internal- by using a style element in the head section
  • External- by using a link element to link to an external CSS file
